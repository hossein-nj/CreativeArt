<h2>🔧 How to run project</h2>

First of all, clone the project to your working directory by running the following command:

```bash
git clone https://github.com/hossein-nj/CreativeArt.git
```

Then install the dependencies:

```bash
npm install
```

Now run the project: 

```bash
npm run t
```
